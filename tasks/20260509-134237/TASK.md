# Improve logging in the project

- STATUS: IN_PROGRESS
- PRIORITY: 70
- TAGS: chore

I would like to have nicer logging in the project. I want to see more useful
information. Even if it's in debug mode. I would like to see when tools are
getting called, what their output is etc etc. These would be nice to be in
"DEBUG" mode only. Also being able to see all the traces from the CLI tool
would be extremly helpful. I would like to be able to see when the main model
asks for another sub agent to do something. When an agent asks a tool for info.
How the tool processes the input and what the output of the tool is. What the
output of each sub agent is and what the main agent gets back. Maybe even how
long each call took or how much RAM or stuff like that. Have really nice
diagnostics of what is happening on each call to different tools and agents.
